# orbis-geonet/.github

This is the special `.github` repository for the **Orbis Geographical Internet (GeoNet)** organization. Its [`profile/README.md`](./profile/README.md) is what renders on the organization landing page at **[github.com/orbis-geonet](https://github.com/orbis-geonet)** — the introduction to Orbis, the clone-operator onboarding, the on-chain economics, and the clone registry.

- **Organization landing page:** [github.com/orbis-geonet](https://github.com/orbis-geonet)
- **Org profile source:** [`profile/README.md`](./profile/README.md)

## Running your own Orbis clone?

Fork the stack (iOS, Android, web, backend, the Solana smart contract), rebrand it, and federate it into the orbis-geonet. To have your clone listed in the registry, open a pull request or email [info@geonetinternet.com](mailto:info@geonetinternet.com).
